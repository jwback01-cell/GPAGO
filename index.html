네이버 API로 자동 가져오기 권장
대기중
⚙️ API 사용 조건 (클릭해서 보기)
  1. "키워드 순위 → 설정" 탭에서 네이버 쇼핑 Open API Client ID/Secret 등록
  2. 등록 후 모든 페이지에서 자동으로 API 호출 가능
  3. 네이버 API는 일 25,000회 무료 호출 한도
  4. 네이버가 일시 차단 시 아래 "JSON 직접 붙여넣기" 방식 사용
📋 JSON 직접 붙여넣기 (수동 복사용)
복사 방법: F12 → Network → 필터 all?sort= → 페이지 새로고침 → 요청 클릭 → Response 우클릭 Copy value

순서고정 가중치

구문별로 매치 상품 수 카운트 (공백 제거 부분일치)
구문을 입력하고 "매치 분석 실행"을 눌러주세요.

위치 가중치

두 단어 사이 최소 단어 거리 계산
거리 = 입력 단어들을 감싸는 최소 구간에서 단어들 사이에 낀 단어 수 (0이면 모두 붙음)
단어 2개 이상 입력 후 실행해주세요.

상품명 가중치

띄어쓰기 단위로 각 단어의 포함 상품 수 카운트
검색어를 입력하고 실행해주세요.

키워드 분해

키워드 + 검색수 붙여넣으면 겹치는 부분을 찾아 코드(A,B,C…)로 분해
💡 구분자: 탭, 쉼표, 연속 공백 모두 지원 (엑셀 → 복사 → 여기 붙여넣기 권장).
검색수는 숫자만 추출 (쉼표/단위 무시). 검색수가 없으면 0 처리.
붙여넣기 후 "분해 실행"을 눌러주세요.
